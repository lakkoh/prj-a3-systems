## Цели проекта

Цель проекта: полное обновление программного обеспечения, используемого в компании, создание унифицированной системы управления и развёртывания операционных систем, пакетов программ и драйверов. Внедрение политик безопасности, антивирусов, продуманных ACL-прав, а в перспективе - домена.

## ОС

### Принципы построения и распространения ОС и ПО

Основная задача проекта - унификация систем, централизация управления и упращение развёртывания. А кроме того, - выстраивание системы прав доступа. В соответствии с этими требованиями предлагается следующее.

**Развёртывание и обновление.** Всё ПО, а также некоторые настройки будут поставляться в виде пакетов. В качестве пакетного менеджера будет использоваться Chocolatey. Всё одобренное ПО и настройки собираются в пакеты для автоматической установки и располагаются на частном сервере. Вместе с конкретными пакетами, будут составляться метапакеты (группы) для конкретных редакций систем, оборудования. Базовая система будет представлять из себя почти голую Windows с предустановленными пакетным менеджером и другими программами для администрирования: ПО, драйвера и настройки будут поставляться с помощью chocolatey, домена (в будущем) и скриптов развёртывания по типу autounatend.

**Централизация управления.** В качестве основной системы управления всеми машинами будет использоваться Tactical RMM. Клиенты для системы будут предустановлены в базовом образе. Централизация управления - важная часть, однако будет внедрена на последних этапах.

**Унификация систем.** Чтобы гарантировать стабильную работу, наличие последних обновлений, конфигурации всех машин должны быть унифицированы: программно и операционно. Операционная унификация - это внедрение ОС на одинаковых базовых сборках. Программная унификация - это внедрение унифицированных пакетов Chocolatey с одинаковым набором ПО для одинаковых редакций систем. В случае, если внедрение новой ОС осложненно (несовместимость оборудования, сложность настройки и т.п.), должна быть реализована хотя бы программная унификация.

### Базовые сборки

Базовые сборки будут созданы вручную из официалных образов Microsoft, иные сборки запрещены из соображений безопасности. В них будут предустановлены программы для администрирования, отключено ненужное ПО, сконфигурированы некоторые настройки. Всего будет создано 3 базовые сборки, представленные в таблице

| Код              | Описание                                                                                                                | На базе сборки |
| ---------------- | ----------------------------------------------------------------------------------------------------------------------- | -------------- |
| **`10_a3_base`** | Основная базовая сборка Window 10 для менеджеровского и клиентского оборудования                                        | 22H2           |
| **`7_a3_base`**  | Дополнительная базовая сборка Windows 7 для устаревшего оборудования *(оборудование необходимо заменить в перспективе)* | -              |
| `xp_a3_base`     | Специальная сборка для работы с устаревшим печатным оборудованием для совместимости драйверов                           | -              |

### Редакции ОС 

Редакция ОС соответствует роли машины в компании. Каждая редакция предполагает свои настройки безопасности, ПО и драйвера. Под каждую редакцию пишется спецификация, под спецификацию составляются пакеты и метапакеты chocolatey. При этом жёсткая привязка к редакции не предполагается, фактически же роли машин будут гибкими: базовая редакция/спецификация нужна на этапе развёртывания, но машине можно присвоить дополнительные роли, включающую соответствующие настройки, ПО и драйверы. Например, машина настраивается как клиентская с базовой редакцией 10_a3_client, но для печати единым пакетом ему ей предоставляется ПО, драйвера и настройки для работы с принтером.

| Редакция             | Описание                                 |
| -------------------- | ---------------------------------------- |
| `10_a3_universal`    | Универсальная редакция (Клиент/Менеджер) |
| `10_a3_checkout`     | Сборка для кассы                         |
| `10_a3_checkout_ext` | Расширенная сборка для кассы             |
| `10_a3_client`       | Только клиентские места                  |
| `7_a3_universal`     | Универсальная сборка на Windows 7        |
| `7_a3_clients`       | Только клиентские места на Windows 7     |
