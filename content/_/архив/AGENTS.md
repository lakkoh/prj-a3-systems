# AGENTS.md — про.а3.системы

## Язык

Все документы на русском языке. Проект российский, заказчик — компания А3.

## Что это за проект

Проектирование архитектуры спецификаций, пакетов/сборок и развёртывания Windows на компьютерах компании А3. Последовательность: **спецификация → пакет/сборка → установка**. Распространение версий — ручное (схема будет уточнена после спецификаций). Скриптинг — второстепенное, на основе спецификаций.

## Ключевые соглашения

Именование файлов — **стандарт ВС 2** (`стандарт.вс.2.md`).

Ключевые идентификаторы:

| Сущность | Формат | Пример |
|---|---|---|
| Образ ОС | `img.<arch>.<os>.<build_edition>.<version>.<lang>.<ext>` | `img.amd64.windows.10_a3_universal.v2026-06-09.ru.iso` |
| Спецификация | `spc.any.<redaction>.<version>.<lang>.md` | `spc.any.10_a3_universal.v2026-06-01.ru.md` |
| Программа | `prg.<arch>.<vendor_product>.<edition>.<version>.<lang>.<ext>` | `prg.win64.corel_draw.corporate.v24-2-1-446.ru.msi` |

Версия спецификаций — ISO8601: `v20260601`, `v20260609`.

## Разрабатываемые редакции

- 10_a3_universal - Универсальная сборка (Клиент/Менеджер)

- 7_a3_universal - Универсальная сборка на базе windows 7 (для старых машин)
